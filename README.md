# 🛰️ VALOSINT CHECKER ID

**VALOSINT CHECKER ID** adalah tool audit keamanan akun berbasis Python yang dirancang untuk pengujian penetrasi legal dan edukasi. Dilengkapi dengan fitur rotasi proxy otomatis dan penanganan token CSRF untuk simulasi login yang akurat.

## 🚀 Fitur Utama
- **Satelit Scanning Animation:** Tampilan UI terminal yang interaktif.
- **Proxy Rotation:** Menggunakan 1 proxy per 1 request untuk anonimitas.
- **Multi-threaded:** Mengecek ratusan akun dalam hitungan detik.
- **Auto-Save:** Hasil valid otomatis tersimpan di folder `results/`.
- **Live Timestamp:** Log waktu real-time untuk setiap percobaan login.
